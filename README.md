# 📚 Student Manager CLI

A Vietnamese CLI-based Student Grade Manager built with Python OOP.
Manage students, subjects, GPA and academic ranking — all from your terminal.

## ✅ Features
- Add / Delete / Search students
- Auto-save to JSON on every action
- GPA calculation (10-point Vietnamese scale)
- Academic ranking (Xuất Sắc / Giỏi / Khá / Trung Bình / Yếu)

## 🚧 Incoming
- Subject & Grade Management (pick from 20-30 popular Vietnamese subjects)
- Colorama colored terminal output
- Edit student info
- Export to .txt / .csv

## 🛠️ Built With
- Python 3.x
- JSON (data storage)
- OOP (Student & Manager classes)
